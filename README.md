🚦 Triple Riding & No Helmet Detection System

AI-based computer vision system developed to detect traffic rule violations such as triple riding and riders without helmets on two-wheelers. Built using deep learning and real-time video/image processing techniques.

Two-Wheeler Violation Detection System | Computer Vision | Python | YOLO | OpenCV
📌 Project Overview

This project focuses on improving road safety enforcement by automatically identifying violations in two-wheelers using AI.

The system detects:

👥 Triple riding (more than two people on a bike)
🪖 Riders not wearing helmets

It can be applied in:

Smart traffic monitoring systems
Law enforcement automation
CCTV-based surveillance

✨ Features
🚀 Real-time detection using deep learning models
🧠 YOLO-based object detection for high accuracy
🎥 Works with images, videos, and live camera feeds
🪖 Helmet detection for rider safety compliance
👥 Triple riding detection using person counting
📊 Scalable for smart city applications

Tech Stack
Python
OpenCV – Image & video processing
YOLO (You Only Look Once) – Object detection
NumPy / Pandas – Data handling
Deep Learning Frameworks (TensorFlow / PyTorch)

Deployment
📋 Prerequisites
Python 3.8+
Install dependencies:
pip install -r requirements.txt
▶️ Run the Project
python detect.py --source "input_video.mp4"

OR for webcam:

python detect.py --source 0
